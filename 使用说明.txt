=== 瞄准小球自动连点器 ===

【使用前准备】
本工具依赖 Python 3 环境和以下库：opencv-python, numpy, pywin32, Pillow
自动安装：双击 "启动.bat" 即可一键安装依赖并启动。

【启动方式】
方法一（推荐）：双击 "启动.bat" → 选择 1(正常) 或 2(慢速调试)
  1 = 正常模式：自动高速点击，F6 启动/停止，F7 停止
  2 = 慢速调试：鼠标平滑移动可目视观察，用于调试定位问题

方法二（打包成 EXE）：
  双击 "打包.bat" → 自动打包 → EXE 会复制到桌面

【使用步骤】
1. 先启动《生死狙击2》游戏，进入"瞄准小球"小游戏
2. 双击 "启动.bat"，选择模式
3. 切回游戏界面，按 F6 开始自动点击，按 F7 停止
4. 也可以直接操作弹出的 GUI 窗口按钮

【热键】
F6 = 启动 / 停止自动点击
F7 = 强制停止

【文件说明】
auto_ball_clicker.py   - 主程序
requirements.txt       - Python 依赖列表
启动.bat               - 一键安装依赖并启动
打包.bat               - 一键打包成 exe
使用说明.txt           - 本文件

【注意事项】
- 建议以管理员权限运行（右键 → 以管理员身份运行启动.bat）
- 仅限在《生死狙击2》游戏窗口内操作，不会误触外部界面
- 如小球识别不准，可编辑 auto_ball_clicker.py 调整 Config 类中的参数
